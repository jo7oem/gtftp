# gtftpd
## 概要
TFTP サーバ、および TFTP クライアントの実装です。
## 関連RFCと実装状況

| RFC番号   | 概要                                                                 | 実装状況   |
|-----------|----------------------------------------------------------------------|------------|
| RFC 1350  | TFTPの改訂版。エラーコードの拡張や仕様の改善。                        | 未実装     |
| RFC 2347  | オプション拡張の導入（TFTP Option Extension）。                       | 未実装     |
| RFC 2348  | ブロックサイズオプションの拡張。                                      | 未実装     |
| RFC 2349  | タイムアウトと転送サイズオプションの拡張。                            | 未実装     |
